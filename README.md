# CIFFile

CIFFile is a Python library for reading, processing, validating, and writing
Crystallographic Information Files ([CIF](https://www.iucr.org/resources/cif)),
including Protein Data Bank Exchange macromolecular Crystallographic Information Files
([PDBx/mmCIF](https://mmcif.wwpdb.org/docs/user-guide/guide.html))
used by the Worldwide Protein Data Bank ([wwPDB](https://www.wwpdb.org/)).


## Installation

CIFFile can be installed from PyPI, e.g., using `pip`:

```
pip install ciffile
```

## Documentation

Detailed documentation is provided in the library.
For a quickstart guide, refer to the
[`QUICKSTART.ipynb`](./QUICKSTART.ipynb) Jupyter notebook.
